Interfaces

IStream Streaming interface.

Classes

File Class for handling filesystem files.
Dir Class for handling filesystem directories. This class will collect information about the requested directory and give easy access to that information. The contents of the directory will be also be collected and can be easily looped through.
IO Abstract IO class
StreamReader The StreamReader class has a number of methods to ease reading of a file.
Stdin Reads from stdin, i.e. the keyboard in most cases.
Options Class for parsing command line options
Argument Pretty much an enum. Only contains constants to be used for stateing if an option needs an argument or not, or if it may have an argument.
Option Class representing an option on the command line
OptionsException Options exception
Image The image class makes it easy to manipulate images.
StringImage Creates an {@see Image} object from a string
Graphics Master class for graphics
GraphicsException Exception
Version Class representing a version, like 1.2, 0.2.345 etc
Date Date class
Object Just an empty object, pretty much as a stdClass
String General string class
StringReader String reader that mimics the {@see StreamReader} class.
Iterator A generic iterator
IteratorOutOfRangeException Generic exception when the iterator is out of range
Cache Simple cache class. Has support for SQLite databases.
CacheResult Cache result class. This class should never be instantiated from the outside.
