\PLib

\PLibDir

Class for handling filesystem directories. This class will collect information about the requested directory and give easy access to that information. The contents of the directory will be also be collected and can be easily looped through.

 $dir = new Dir ('/path/to/dir');
 // Loop through the contents of the directory and show only PHP files.
 // The $f here will be a File object {@link File}
 while ($f = $dir->emit ('*.php')) {
   echo $f->path ($f->nice_size ()) . "<br/>";
 }
